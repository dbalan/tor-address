# Compute Onion V2 address from RSA key

Calcuates `onionv2` address from RSA private key.

## Usage
```
⚡go build
⚡./onion-v2-addres
Usage: ./onion-v2-addres <path-to-priv.pem>
⚡  ./onion-v2-addres test.pem
tor address: testvh5q4jmneo5j.onion
```
